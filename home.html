Relational Database of CVE records with CVSS scores to evaluate cyber risks of popular Relational Database Management Systems

QUERY BOX

Description: This relational database is made to house reliable records detailing the database security of well-known relational database management systems. The database will feature objective criteria and evaluations for several popular RDBMS (Relational Database Management System) and their different versions. These universally accepted criteria will include CVE (Common Vulnerability and Exposure) and CVSS (Common Vulnerability Severity Score) records. The intended purpose of this tool is to help individuals, whether pursuing personal work or representing a company, make informed decisions about which RDBMS they wish to utilize. Given that security is a major deciding factor.
